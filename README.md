# おじさんクエスト（ojisan-quest）

おじさんユニバースの**オリジナルのターン制コマンドRPG**（特定作品のパロディではない／作りやすさ最優先）。
物語前提は「定時に帰れない呪い」——残業の夜、会社ビルに閉じ込められたおじさんが各階の“残業の魔物”を倒し定時退社（脱出）を目指す。
詳細は docs/GAME_DESIGN.md と ~/Developer/games/_starter-kit/。

## 動かす
```
npm install
npm run dev      # http://localhost:5173/
npm run build
```

ルール：有名作の固有キャラ/名称/ロゴ/音楽/見た目は使わない（全部おじさんオリジナル）。git identity は匿名固定（hoikuen）・コミット前確認。
